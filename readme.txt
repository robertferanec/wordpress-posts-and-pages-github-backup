Backups WordPress content of your posts and pages to GitHub

== Description ==

This plugin backups the content which you write on the Edit page. It will upload it to GitHub.

When you are writing a page/post and you would like to backup the text, you can use this plugin. If anything happen to your server, you always will have the texts on GitHub.

This plugin can be also used for collaboration. You can easily see and track any changes done by other people + you can comment the changes on GitHub.


== Installation ==

1. Upload the plugin files to the WordPress plugin directory (e.g. '/usr/local/apache2/htdocs/wp-content/plugins/') or clone it from our github:

        cd /usr/local/apache2/htdocs/wp-content/plugins/
        git clone https://github.com/robertferanec/wordpress-posts-and-pages-github-backup.git

2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Admin->Posts Backup to configure the plugin
